There are generally 9 types of common flooring patterns that is able to be laid using vinyl tiles.


Grid

  • It is also referred as stack bonds where tiles are stacked vertically and horizontally aligned with each other.
Offset
  • Commonly referred as running bonds. This layout is laid half of the width of each tiles connecting each other.
Stagger
  • It is an alternative arrangement of tiles in a wavy line. Mostly done in 2 to 3 colours.
Diagonal
  • A pattern laid 45 degrees instead of a horizontal straight line.
Herringbone
  • The pattern is named Herringbone after a fish call Herring fish due to its skeleton structure. It is seen as a zigzag pattern. Some would also call it as broken twill weave.
Weave
  • Also referred to as basketweave, this variation combination with different colours will produce different patterns.
Corridor
  • It is a grid pattern layout with rows intervening throughout.
Solder
  • It is a stack of rectangular tiles with an alternate direction variation.
Random
  • It is an arrangement of rectangular tiles in no particular order.
